## Getting Started

Follow these steps to run the application locally:

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/prajesh9921/NoteTaker/tree/backend
   ```

2. Install Dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
